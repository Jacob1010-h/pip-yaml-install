# YAML Package Installer

A Python script that installs packages from a YAML file using pip.
Works on Windows, Linux, and macOS.

## Requirements

- Python 3.6+
- pip
- PyYAML

## Setup

1. Install PyYAML if you don't have it:

   ```bash
   pip install pyyaml
   ```
